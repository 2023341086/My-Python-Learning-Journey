# My-Python-Learning-Journey
## 这是我的Python程序学习库^_^
*如果你要运行这些函数模块，请自行下载调用，包含两个函数*
### ❤第一个函数是 [*scores_statistic*](scores_statistic.py) ，它的作用是帮助老师计算班级的学生成绩。
- 当你输入十个成绩后，可以帮你输出你输入的成绩
- 可以帮你统计输入的成绩总值
- 可以帮你计算出输入的成绩的最大值
- 可以帮你检测出所有不及格的成绩
### ❤第二个函数是 [*fruit_price*](fruit_price.py) ，它的作用是帮助水果店店员或来到水果店的顾客查询店内水果的价格。
- 运行程序后会显示可选水果和未上架水果，只有可选水果能查询价格
- 按要求输入列表当中的水果即可
- 如果还需重新查询，只需输入 *fruit_price()* 重新运行该程序
